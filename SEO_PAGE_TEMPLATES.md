# SEO_PAGE_TEMPLATES.md

## Purpose
Provide repeatable SEO page templates for Procitec.io.

## Template 1: Product Page
Use for:
- EAM
- EMS

### Structure
1. Title
2. Intro
3. Problem
4. Solution overview
5. Key capabilities
6. Benefits
7. Use cases / industries
8. FAQ
9. CTA

### Example Title
`Enterprise Asset Management Software for Industrial Plants | Procitec`

---

## Template 2: Problem / Intent Page
Use for:
- preventive maintenance software
- asset tracking software
- energy monitoring software

### Structure
1. H1 with clear search intent
2. problem definition
3. who it is for
4. solution explanation
5. product fit
6. benefits
7. FAQ
8. CTA

### Example
`Preventive Maintenance Software for Manufacturing Plants | Procitec`

---

## Template 3: Industry Page
Use for:
- manufacturing
- utilities
- infrastructure

### Structure
1. industry pain points
2. where Procitec fits
3. application relevance
4. use cases
5. deployment suitability
6. FAQ
7. CTA

## Metadata Rules
Every page should define:
- title
- description
- canonical if needed
- open graph title / description

## Content Rules
- one page should serve one intent
- use headings clearly
- include FAQ
- link to product and related pages
- avoid keyword stuffing
