# Design Rules

## Visual Style
- Clean white/light neutral backgrounds
- Strong typography and spacing
- Premium B2B industrial feel

## Colors
- Primary: #8b3dff
- Secondary: #e1511b (use sparingly)

## Layout
- Use grid-based layout
- Maintain consistent spacing
- Avoid clutter

## Components
- Reusable components
- Avoid excessive rounded corners
- Keep shadows soft and subtle

## Motion
- Minimal animations
- Smooth and purposeful transitions

## Responsiveness
- Mobile-first approach
- Clean tablet and desktop scaling
