# CTA_STRATEGY.md

## Purpose
Standardize calls to action across Procitec.io.

## CTA Principles
- direct
- clear
- action-oriented
- low ambiguity
- aligned with user intent

## Primary CTAs
Use for high-intent actions:
- Open EAM
- Book Demo
- Talk to Expert

## Secondary CTAs
Use for lower-friction navigation:
- View Applications
- Explore Platform
- Read Support Guide

## CTA Placement Rules
- above the fold in hero
- after major explanation blocks
- near the end of the page
- in product cards where appropriate

## CTA Copy Rules
Prefer:
- Open EAM
- View Applications
- Book Demo
- Talk to Expert

Avoid:
- Click Here
- Learn More (unless context demands it)
- Submit
- Discover More

## Visual Rules
- primary CTA should be visually dominant
- secondary CTA should be clear but quieter
- avoid too many competing CTAs in one block
- buttons should follow standardized variants

## Mobile Rules
- CTA should remain visible early
- large tap targets
- avoid tiny inline text links for critical actions
